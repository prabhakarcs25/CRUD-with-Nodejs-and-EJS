# Node.js CRUD Application (EJS)

A simple CRUD (Create, Read, Update, Delete) web application built using **Node.js**, **Express.js**, and **EJS** template engine.  
This project demonstrates server-side rendering, RESTful routing, and basic database operations.

---

## Features
- Create, read, update, and delete records
- Server-side rendering using EJS
- RESTful routes with Express
- Clean MVC-style structure
- Form handling and validation
- Basic error handling

---

## Tech Stack
- Node.js
- Express.js
- EJS (Template Engine)
- MongoDB / MySQL (choose one)
- Mongoose / mysql2
- HTML, CSS, Bootstrap
